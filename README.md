# Focus Nuke

Transform YouTube into an intentional, distraction-free environment.

## **Purpose:** Remove distractions from YouTube.

### **Features:**
- Distraction toggle (Nuke Mode)
- OLED dark mode (togglable)
- Custom accent colors (currently only Turqoise)
- Note-taking integration (multiple notes per video, saves data locally)


### **WIP:**
- Super Nuke, gets rid of everything but the video, and makes playlists more compact (experimental, default off)
- More themes, custom themes
- Edge cases, bug fixes
- Make the logo and profile buttons more aesthetic
- Make notes feel easier to take and use
- Notes collection (place to see all your notes)
- Video bookmark integration
- Button to go back to where you left off last in a course or playlist

### **Bugs:**
- none at the moment
